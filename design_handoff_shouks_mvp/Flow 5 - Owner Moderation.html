Want listings to go live instantly?
Set up auto-approval rules — e.g., members with 3+ prior approved listings skip review. You can always spot-check.
NotesListing cards share the same vocabulary as the marketplace browse view so owners recognize them instantly — but they get a bold top-edge accent on edited ones and a diff chip explaining what changed. This solves the #1 owner question: "what's different?" Bid/view counts warn the owner that a rejected edit has real consequences for a live listing. The auto-approval callout is the long-term pressure release for growing communities.

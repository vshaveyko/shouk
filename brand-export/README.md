# Shouks Brand Package · v1

Drop this folder into your repo or share it with your team.

## Start here

- **`Brand Guidelines (standalone).html`** — the full visual brand book. Single file, works offline, drag into any browser.
- **`Implementation Guide (standalone).html`** — step-by-step for engineers: what to copy, where, how to wire it up. Single file, works offline.

## For engineers

- **`BRAND.md`** — the rules in plain text. Link this from your main README.
- **`tokens.json`** — all design tokens (W3C format). Import into CSS, Tailwind, or Style Dictionary.
- **`dist/`** — logos, icons, favicon. Copy the files you need into `/public/` or `/src/assets/`.

## For designers / reference

- **`Brand Guidelines.html`** — same as the standalone, but loads fonts from Google Fonts (smaller file, needs internet).
- **`Implementation Guide.html`** — same pattern for the engineering guide.
